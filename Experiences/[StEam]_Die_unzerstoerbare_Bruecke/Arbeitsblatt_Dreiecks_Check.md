# Arbeitsblatt: Dreiecks-Check (Forces-Diagnose)

## Ziel
Nach dem Belastungstest findest du heraus, welche Teile deiner Brücke auf **Zug** und welche auf **Druck** belastet wurden. So erkennst du, wie Dreiecke die Kräfte verteilen.

## Material
- Foto deiner Brücke (vor dem Test)
- Zwei Filzstifte oder Marker:
  - 🔴 Rot = Druck (Stäbe werden zusammengedrückt)
  - 🔵 Blau = Zug (Stäbe werden auseinandergezogen)

## Schritt 1 – Beobachten
| Frage | Antwort |
| --- | --- |
| Wo hat deine Brücke zuerst angefangen zu knicken/brechen? | |
| Welche Dreiecke sind stabil geblieben? | |
| Gab es rechteckige Bereiche ohne Diagonale? Was ist dort passiert? | |

## Schritt 2 – Markieren
1. Druckzonen rot markieren.
2. Zugzonen blau markieren.
3. Schreibe dazu, ob dein Truss aus gleich großen oder gemischten Dreiecken bestand.

## Schritt 3 – Rückblick
- Welche Verstärkungen würdest du beim nächsten Mal zuerst einbauen? ______________________
- Was hat das Labor/Science-Buddies-Experiment über Dreiecke bestätigt? __________________
- Welche reale Brücke (aus der Galerie) verhielt sich ähnlich? ___________________________

## Bonus
Wenn Zeit bleibt, nutze Spaghetti-Reste und zeige deinem Team, wie ein Rechteck ohne Diagonale einknickt. Füge dann eine Diagonale hinzu und teste erneut.
